INTERACTIVE DISNEY MUSEUM

This web page was created using HTML and Javascript. The Interactive Disney Museum showcases a timeline of Disney movies. Each movie offers a different interaction. The goal was to provide a fun and interesting way of learning about Disney without just reading a Wikipedia page. The overall theme was to to make it minimalistic and clean so that all movies blend better together.

This web page currently works on Safari, Chrome, and Firefox. Please adjust window/zoom accordingly to account for various screen sizes.

If objects seem to be shifted, increase or decrease the zoom until things are aligned properly. (Example: Zoom at 90%/100%/110%)
